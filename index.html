1. Download and open Visual Studio IDE software. 2. Create a new folder on your computer. 3. Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created in step 2. 4. Install Live-Server Extension if not already installed.

The console screen is used for displaying text on the output screen as holders. How to open on output screen: 1. Double click and click on "Inspect". 2. At the top, click on "Console". 3. You are at the console screen!
